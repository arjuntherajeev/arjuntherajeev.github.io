About
Diary of Arjun

My journey from System.out.println("Hello World"); to print("Hello World") and beyond


I picked up programming in the eleventh grade. It was a glorious time as I learnt all about Object Oriented Programming (OOP) with the syntax of C++. Days turned into nights and I was coding away, learning about if-else conditionals, for loops, and #include statements.

Fast forward to University where I studied Computer Systems. My mind was blown everyday because during this time, I met C and Java.

It was mind-blowing because I discovered two new syntaxes, two new IDEs (Eclipse for Java and the Linux Terminal for C), and finally understood the difference between a high-level and low-level programming language. I also discovered Android where Java code magically creates Android APKs which run as Apps on my Moto G. I was hooked! In University, I also had the pleasure of discovering a variety of popular (and esoteric) programming languages including C, Python, MIPS Assembly, ARM Assembly, and Prolog.

I cannot understate my experimentation with Web technologies. While I started with vanilla HTML, CSS, and JavaScript, I quickly evolved to using CSS frameworks such as Bootstrap, MaterializeCSS, and Skeleton. I wanted to go with an exotic backend and was pretty jaded about the usual LAMP stack. That's when I met WEB2PY. Web development with Python. I also dived into a variety of NoSQL and SQL databases especially Neo4j.

As University progressed, I enhanced my knowledge about Networking, Cyber Security, Databases, Data Analysis, Machine Learning, and Information Systems. Throughout this journey, my love for Python grew exponentially because of its versatility.

In my fourth and final year of University, I worked on a unique dissertation project. I built Resistance, an Anti-Malware software powered by Machine Learning. The project involved two parts:

  1. Training ML models to detect Malware. This type of problem is called Binary Classification where the goal is to determine if a PE file is malicious or benign.
  2. Designing and building an Anti-Malware software with an ML engine which can be loaded with an external ML model. The software also required a GUI and other common features such as Signature-based detection and Quarantining

For this project, I worked with Python's scikit-learn module for training ML models and C# for building the Windows software. I had a brilliant experience working with Dr. Hani (my mentor and supervisor) who inspired me to accomplish both the tasks and much more.

Today, I work as a Cyber Security Consultant and Analyst where I help organizations build and fine-tune their Security Operations Centers (SOCs). My role allows me to work with SIEM and SOAR platforms such as IBM QRadar and IBM Resilient as well a whole range of products from various Cyber Security vendors such as Nozomi, CrowdStrike, and Qualys.

Being in the world of Cyber Security, I will always accept that I need to keep on learning. New threats, new vulnerabilities, new technologies, new tools, and new vendors emerge daily. It is my duty as a professional to learn and share the knowledge with the world.

That being said, I welcome you to Diary of Arjun.
